# Project 01 - Kali Linux Home Lab Setup

## Objective

Build a personal cybersecurity lab using VirtualBox and Kali Linux.

## Environment

* Windows 11 Host
* Oracle VirtualBox
* Kali Linux 2026.1

## Tasks Completed

* Installed VirtualBox
* Downloaded Kali Linux Virtual Machine
* Imported Kali into VirtualBox
* Logged into Kali Linux
* Verified network connectivity
* Updated package repositories

## Linux Commands Learned

### hostname

Displays the system hostname.

### whoami

Displays the current logged-in user.

### ip a

Displays network interfaces and IP addresses.

### uname -a

Displays system and kernel information.

### cat /etc/os-release

Displays operating system information.

### sudo apt update

Updates package information from repositories.

## Lessons Learned

I successfully built my first cybersecurity lab and gained hands-on experience with Linux, virtualization, networking basics, and package management.
